# Contributing

Пожалуйста, сначала публикуйте все вопросы и проблемы в `Issues`

## Contribution to the project

1. Есть много способов внести свой вклад:
   - написания руководств;
   - публикаций в блогах;
   - улучшения документации;
   - отправки отчетов об ошибках;
   - MR для реализации совершенно нового функционала.
2. Если Вы хотите внести свой вклад в проект, то можете начать с прочтения [документации](README.md)
